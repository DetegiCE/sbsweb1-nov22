CSS 선택자(selector) - HTML 태그 중에서 스타일을 적용할 태그를 선택하는 방식

- 하위 선택자 (자손 선택자): 특정 요소 아래에 있는 "모든" 조건을 만족하는 요소
section p -> section 태그 아래에 있는 모든 p 태그

- 자식 선택자: 특정 요소 "바로" 아래에 있는 조건을 만족하는 요소
section > p -> section 태그 바로 아래에 있는 p 태그

- 인접 형제 선택자: 특정 요소에 "바로 다음에" 붙어있는 조건을 만족하는 태그
h1 + p -> h1 태그 바로 다음에 p 태그가 있다면

- 형제 선택자: 특정 요소 "다음에 같은 계층에 있는" 조건을 만족하는 모든 태그
h1 ~ p -> h1 과 같은 계층에 있는 다음에 나오는 p 태그

- 속성 선택자: 특정 태그에 대해서 특정 속성값을 가지는 모든 요소
a[href]: a 태그 중에서 href 속성을 갖는 모든 요소
input[required] {
 border: 1px solid red;
}

- 속성 값 선택자: 특정 태그에 대해서 특정 속성이 해당하는 속성 값을 가지는 모든 요소

a[target="_blank"]: a 태그 중에서 target 속성이 "_blank"인 모든 요소
-> 외부 사이트 연결 아이콘 붙일 때

[class ~= "b"]: 모든 요소 중에서 class 속성이 b를 포함한 모든 요소

[title |= "ab"]: title 속성이 "ab" 이거나 "ab-"로 시작하는 모든 요소

[lang ^= "en"]: lang 속성이 "en"으로 시작하는 모든 요소

[lang $= "asia"]: lang 속성이 "asia"로 끝나는 모든 요소

[href *= "google"]: href 속성에서 "google"을 포함하는 모든 요소

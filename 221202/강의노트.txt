가상 클래스 - 사용자 동작에 의해서 반응하는 속성
link - a 태그를 이용했을 때 링크가 걸리면 파란색 + 밑줄 -> 바꿔주는 속성
visited - a 태그를 이용했을 떄 링크가 이미 방문한 상태라면 보라색 + 밑줄 -> 바꿔주는 속성
hover - 요소에 마우스 포인터를 올렸을 때에 대한 스타일 적용
active - 요소를 활성화 했을 때에 대한 스타일 적용
focus - 요소에 초점이 맞춰졌을 때에 대한 스타일 적용 (마우스 포인터가 해당 위치로 이동했거나, Tab 키를 눌러서 해당 위치로 이동한 경우)

요소의 상태에 따른 가상 선택자 클래스 
target - a태그에 연결된 링크에 대한 스타일
enabled, disabled - 요소에 대한 사용 가능 여부에 따라서 스타일 적용 (사용 가능 -> enabled, 사용 불가능 -> disabled)
checked - 라디오 버튼이나 체크박스 등에서 체크를 한 경우 -> 체크 된 요소에 대한 스타일 부여
not - 아니다 -> 해당 속성이 아닌 요소

구조 가상 클래스 - 웹에서 구조를 기준으로 특정 위치에 있는 요소에 스타일 적용
only-child -  유일한 자식 - 자식이 하나일 경우, 그 자식 요소를 선택
A:only-type-of - 유일한 타입 - 부모의 자식 중에서 A 타입이 유일한 경우 그 자식 요소를 선택
first-child - 첫 번째 자식
last-child - 마지막 자식
A:first-of-type - 부모의 자식 중에서 A 타입들 중 첫 번째 자식
A:last-of-type - 부모의 자식 중에서 A 타입들 중 마지막 자식
nth-child(n) - 부모의 n번째 자식
nth-last-child(n) - 부모의 끝에서부터 n번째 자식
A:nth-of-type(n) - 부모의 자식 중에서 A 타입들 중 n번째 자식
A:nth-last-of-type(n) - 부모의 자식 중에서 A 타입들 중 끝에서부터 n번째 자식

위의 괄호 안의 n 자리에 숫자 뿐만 아니라 홀수 짝수도 들어갈 수 있음 -> odd(홀수), even(짝수)
숫자 뿐만 아니라 수식도 가능 (an+b 형태의 수식)
3n+1 -> 1, 4, 7, ...
2n -> 0, 2, 4, 6, ...



가상 요소 - 문서의 특정 부분에 스타일을 가상 요소를 만들어서 추가하기
-> 화면에 특별한 것을 추가하기 위해서 굳이 HTML에 요소를 추가하지 않고 CSS를 이용해서 요소를 추가하는 용도

가상 클래스는 CSS 를 작성할 때 요소:~~ (콜론 1개 사용)
가상 요소는 CSS를 작성할 때 요소::~~ (콜론 2개 사용)

first-line, first-letter - 지정한 요소의 첫 번째 줄이나 첫 번째 글자에 스타일을 적용할 때 사용
last-line, last-letter - 지정한 요소의 마지막 줄이나 마지막 글자에 스타일을 적용할 때 사용

before, after -> 요소의 앞과 뒤에 컨텐츠를 추가
CSS를 이용해서 요소에 텍스트나 이미지 등을 추가할 수 있음
before와 after 가상 요소는 반드시 content 라는 속성을 가지고 있어야 함 -> content가 글씨 등의 컨텐츠를 추가해주는데, 추가할 컨텐츠가 없더라도 빈칸으로라도 추가해 놓아야 함
-> 가상 요소로 추가하게 되면 선택 X
-> 기존 사이트 구조에 영향을 미치지 않음
